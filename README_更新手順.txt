BASE CAMP YAMATO Ver1.0 修正版

【GitHubへのアップロード方法】
1. GitHubの base-camp-yamato リポジトリを開きます。
2. 現在の index.html、style.css、script.js、assets フォルダを、このZIP内の同名ファイルで置き換えます。
3. ファイルはZIPのままではなく、解凍して中身をアップロードしてください。
4. Commit changes を押します。
5. 反映後、スマホではURL末尾に ?v=10 を付けて確認してください。

【主な修正】
・スマホでトップ画像の左右が切れないように修正
・トップ画像の固定高さ、最小幅、位置ずれを解除
・スマホ用の実際に押せる「LINEで予約する」「料金を見る」ボタンを追加
・横スクロールを防止
・ヘッダーメニューをスマホ対応
・料金、備品、ペルソナ、アクセスをスマホ1列表示に調整
・画像拡大モーダルを実装
・PC表示も崩れにくいレスポンシブ設計に整理

【LINE予約URLについて】
現在はLINE公式アカウントのURLが未設定です。
URL取得後、index.html内の href="#" または href="#reserve" を実際のLINE URLに変更してください。
