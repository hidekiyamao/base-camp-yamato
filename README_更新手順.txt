今回の修正
- 利用案内画像の禁煙欄を修正
  見出し：「室内・建物周辺は／禁煙です」
  本文：「室内はもちろん、建物周辺での喫煙もご遠慮ください。」

- Googleマップのリンク先を新住所に変更

GitHubへの反映
1. ZIPを解凍
2. 中にある index.html / style.css / script.js / assets フォルダを、
   GitHubリポジトリの最上位にドラッグ＆ドロップ
3. 既存ファイルの上書きを確認して「Commit changes」
4. 1〜3分後に公開ページを Ctrl+F5 で再読み込み

注意
- ZIPファイルそのものをGitHubへ置くのではなく、必ず解凍後の中身をアップロードしてください。
